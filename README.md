📌 Project: Student Form Management

🧑‍💻 This project is a simple Student Form Management system built using HTML, Tailwind CSS, and JavaScript.

✨ Features:

• 👤 Add student details using a form

• 🎂 Input fields for Name and Age

• 🚻 Gender selection using radio buttons

• 📚 Course selection using dropdown menu

• 📧 Email input field for student contact

• 💾 Save button to add student data to the table

• 🗑 Delete button to remove student records

• 🔄 Form resets automatically after saving

🛠 Technologies Used:

• HTML5 – structure of the form and table

• Tailwind CSS – styling and layout

• JavaScript – DOM manipulation and dynamic table updates

⚙️ How it works:

1️⃣ User enters student details in the form

2️⃣ Clicks the Save button

3️⃣ JavaScript creates a new row in the table

4️⃣ Student details are displayed in the table

5️⃣ Delete button removes the student record from the table

🎯 Purpose of the Project:

This project helps practice JavaScript DOM manipulation, dynamic table creation, form handling, and interactive UI development using Tailwind CSS.
